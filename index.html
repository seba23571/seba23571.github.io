PSV - Programa Stock Ventas

Aumenta la productividad de tu negocio con un sistema de gestión comercial fácil de usar, rápido y seguro.

Características

Módulo de carga

Cuenta con una interfaz de carga de productos con ayuda automática de texto predictivo para ingresar datos relacionados de stock, evitando tipeo redundante. Funciones para cálculos de porcentajes sobre precios de los productos, descuentos, remarcación e IVA. Control de inventario de productos faltantes. Control de fecha de caducidad sobre el inventario de productos que se van a vender. Módulo de carga de Productos elaborados. Permite mediante un código de barra estándar o personalizado asociado a un nombre del producto vincular a varios productos para elaborar una promoción o artículos agrupados.

Módulo de ventas

Este módulo tiene una cómoda interfaz de venta para el operador o cajero, detallando la suma de los productos ingresados y el vuelto que corresponde al cliente a través de un lector de códigos de barra. Permite representar la cantidad de productos que están a la venta simplificando productos repetidos. Permite ventas de productos elaborados como promociones con códigos de barras personalizados. Permite buscar productos individuales por nombre o códigos de barra. Permite buscar productos agrupados por nombre o códigos de barra.

Módulo de reportes

Tiene funciones para ver reportes de ventas de diferentes secciones como artículos de mercado, sección carnicería, fiambrería, etc. Permite calcular el reporte de ventas entre diferentes fechas como: semanas, meses y años. Funciones de conexiones de la aplicación. Permite la conexión entre cliente (Operador o Cajero) y servidor (Administrador general) a través de cable lan o wifi en un área de trabajo para carga de datos y ver reportes. Permite la conexión remota entre cliente y servidor fuera del área de trabajo para carga de datos y ver reportes.

Módulo de la interfaz de usuario

Cuenta con tres tipos de privilegios de usuarios de acceso a la aplicación: Administrador General, Administrador (Encargado) y Operador (Cajero). Administrador General: tiene acceso a todas las configuraciones del sistema como establecer conexiones cliente servidor, cambiar precios,ver reportes, configuración de driver fiscal epson y puede dar de alta a otros usuarios o eliminarlos. Administrador: pude hacer ventas, modificación de stock y ver reportes (no puede agregar otros usuarios ni establecer conexión cliente servidor entre dos computadoras). Operador : Solo puede realizar ventas.

El programa permite hacer una copia de respaldo automática de toda la información en el disco local o en la nube.

Opciones adicionales

Podemos agregar las cajas o puntos de venta necesarios para tu negocio y la posibilidad de controlar en los reportes las ventas que se hicieron en cada punto de venta.

Integración con balanzas Systel para para luego tener un control de ventas y ver reportes de diferentes secciones como artículos de mercado, sección carnicería, fiambrería, etc..

El sistema PSV funciona con impresoras fiscales. Con computadoras que tengan un puerto PCI RS-232. Notebooks y PCs con puertos USB deben hacer una prueba de compatibilidad con una licencia de prueba.

Aplicación android (telefonos o tablets) que se utiliza para: ventnder, modificar o eliminar precios, agregar nuevos usuarios y ver reportes sin la necesidad de estar en la caja o punto de venta.

Descarga nuestro programa y probalo GRATIS

1Descarga el programa

Descomprimir el archivo InstaladorPSV.zip. Veras un archivo InstaladorPSV.exe con icono de PSV; hace click con el boton derecho del raton sobre ese icono y elegir la opción instalar como administrador en la pestaña emergente. Después click en siguiente.

2Conecta tu lector de codigos de barra

También podes ingresar los códigos de forma manual (8 o 20 caracteres, ej: jabon123).

3¡Empeza a usarlo!

Para más información hace click en el icono de Youtube que se encuentra abajo o comunicate con nosotros.

Funciona en Windows 10, 8.1, 8, 7, Vista y XP (sp1) , incluidas las versiones de 32 y 64 bits..
